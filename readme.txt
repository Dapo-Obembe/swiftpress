SwiftPress is a modern, lightweight WordPress Full Site Editing (FSE) theme designed for speed and flexibility. 
It provides global styles, patterns, and block templates that let you build and customize your entire site visually in the Site Editor.

== Description ==

SwiftPress is a clean and performance-oriented block theme for WordPress.  
With Full Site Editing support, you can edit headers, footers, templates, and global styles directly from the Site Editor without touching code.

The theme ships with:
- Predefined style variations.
- Custom block patterns for layouts and sections.
- Typography and spacing presets.
- A flexible color palette optimized for readability and contrast.
- Accessibility-ready markup.

Perfect for bloggers, small businesses, portfolio sites, and anyone who wants a fast-loading, future-ready WordPress theme.

== Features ==

* **Full Site Editing** – edit every part of your site with blocks.  
* **Block Patterns** – ready-made section templates to speed up your design process.  
* **Global Styles** – consistent typography, colors, and spacing across your site.  
* **Accessibility Ready** – built with accessibility best practices.  
* **Responsive Design** – looks great on mobile, tablet, and desktop.  
* **Performance First** – lightweight, minimal dependencies, optimized for speed.  

== Installation ==

1. Upload the `swiftpress` folder to the `/wp-content/themes/` directory, or install the theme directly from the WordPress admin under **Appearance > Themes > Add New**.
2. Activate the theme through the **Appearance > Themes** menu.
3. Open the **Site Editor (Appearance > Editor)** to start customizing templates, global styles, and block patterns.

== Frequently Asked Questions ==

= Does SwiftPress support Full Site Editing? =  
Yes. SwiftPress is built as a block theme and fully supports the Site Editor introduced in WordPress 6.3.

= Can I use classic widgets with this theme? =  
No. As a block theme, SwiftPress uses block-based widget areas and patterns instead of classic widgets.

= Where can I find block patterns? =  
You can find custom patterns under the inserter (**+** button) inside the Block Editor or Site Editor.

= Is this theme translation ready? =  
Yes. SwiftPress includes translation files and can be translated into any language using standard WordPress translation methods.
